# 试题图形化解答器

这是一个离线可用的本地单页程序。学生或老师可以输入一道题，也可以输入一整张小卷，程序会自动拆题、识别学科，并生成分步解答、关键公式、答案和 SVG 图形化说明。

## 使用方式

1. 打开 `index.html`。
2. 在左侧输入试题或试卷。
3. 点击“生成图形化解答”。
4. 可复制文字解答，或使用“打印 / 保存 PDF”导出。

## 当前支持较好的题型

- 数学：一次函数与反比例函数交点，自动计算坐标并绘制函数图像。
- 物理：光滑斜面受力分析，自动计算重力分力并绘制受力图。
- 化学：高锰酸钾制氧气、实验室制二氧化碳、乙醇/乙酸结构示意。

## 多题输入格式

可以用空行分隔，也可以用题号分隔：

```text
1. 已知一次函数 y=x+2 与反比例函数 y=4/x 交于 A、B 两点，求交点坐标并画图。

2. 质量为 2kg 的物块放在倾角 30° 的光滑斜面上，求重力分力。
```

## 文件说明

- `index.html`：页面结构。
- `styles.css`：界面样式和打印样式。
- `app.js`：拆题、识别、计算、绘图和复制逻辑。
